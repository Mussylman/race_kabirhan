## ЭТАП 2: Новый Color Classifier

### Вариант A: CLIP/SigLIP Zero-Shot (НОВЫЙ, рекомендуется как первый шаг)
- **Не требует обучающих данных** — работает из коробки
- Используем text prompts: "a horse jockey wearing {color} clothing"
- Модели: CLIP ViT-B/32 (151M) или SigLIP 2 ViT-B (86M)
- Устойчив к изменениям освещения (pretrained на 400M+ пар)
- Реализовано: `pipeline/clip_color_classifier.py`
- Включение: `use_clip=True` в AnalysisLoop или auto-fallback если нет .pt модели
- **Ансамбль CLIP + HSV** для максимальной надёжности

### Вариант B: CLIP Fine-tuned Linear Probe
- Замораживаем CLIP backbone, обучаем только линейный классификатор сверху
- Нужно всего 50-100 кропов на класс (vs 200+ для CNN)
- Скрипт: `tools/train_clip_classifier.py`
- **Лучший вариант когда данные собраны** (~20 эпох, <5 мин GPU)

### Вариант C: MobileNetV3-Small
- Pretrained ImageNet → fine-tune на наши 3 класса
- ~2.5M параметров — ещё влезает в RTX 3060
- TRT FP16 inference ~0.3ms per crop
- Нужно 200+ кропов на класс

### Вариант D: EfficientNet-V2-S (текущий)
- 21M параметров — избыточен для 3 цветов
- Переобучается при малом датасете
- Оставить как fallback

**Рекомендация: Вариант A (CLIP zero-shot) сейчас → Вариант B (fine-tuned) когда данные собраны**

### Метрики для валидации:
- Per-class accuracy ≥ 95% (для каждого из 3 классов)
- Confusion matrix: green vs purple vs yellow без путаницы
- Тест на кропах из каждой камеры (разные углы/освещение)

## ЭТАП 5 (НОВЫЙ): Jockey ReID — идентификация по фото с телефона

**Принципиально новый подход**: вместо определения цвета — распознавание жокея
по визуальным эмбеддингам. Снимаем каждого жокея на телефон 10-15 раз →
модель находит их на камерах ипподрома.

### Почему ReID лучше цвета:
- Не зависит от освещения (CLIP обучен на 400M+ изображений)
- Различает жокеев даже в похожих цветах
- Использует всю визуальную информацию (паттерн, текстура, форма тела)
- 15 фотографий достаточно для надёжного распознавания
- Не нужна ручная разметка цветов

### Реализация:
- `pipeline/jockey_reid.py` — модуль ReID (CLIP / DINOv2 / OSNet + FAISS)
- `tools/build_gallery.py` — создание галереи из фото телефона

### Как использовать:
```
# 1. Создать галерею:
mkdir -p gallery/Аслан_зелёный gallery/Марат_красный gallery/Болат_жёлтый

# 2. Скопировать фото с телефона (10-15 шт каждого жокея)

# 3. Построить эмбеддинги:
python tools/build_gallery.py --gallery gallery/

# 4. Запуск с ReID:
analyzer = AnalysisLoop(cam_mgr, frame_src, use_reid=True, reid_gallery="gallery/")
```

### Бэкенды:
| Модель | Размер | Точность | Скорость |
|--------|--------|----------|----------|
| CLIP ViT-B/32 | 151M | Хорошая | ~5ms/crop |
| DINOv2 ViT-S/14 | 86M | Лучшая для визуальных фич | ~3ms/crop |
| OSNet-AIN | 2.2M | Лучшая для ReID (pretrained) | ~1ms/crop |
