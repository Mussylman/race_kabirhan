# Race Vision — Краткое описание проекта

**Race Vision** — система отслеживания позиций лошадей в реальном времени для Казахстанской федерации конного спорта. Определяет цвета жокейских форм с камер ипподрома и транслирует текущий расклад гонки на веб-табло.

## Стек технологий

| Слой | Технологии |
|------|-----------|
| **Бэкенд** | Python, FastAPI, PyTorch, YOLOv8, OpenCV |
| **Фронтенд** | React 19, TypeScript, Vite, Zustand, Framer Motion, Tailwind CSS |
| **Инфраструктура** | Docker, NVIDIA DeepStream (опц.), go2rtc, Nginx |

## Как работает

```
25 аналитических камер (RTSP)  →  YOLOv8 + CNN  →  позиции жокеев
         ↓                                              ↓
  (видео скрыто от зрителей)                   WebSocket каждые 200мс
                                                        ↓
4 PTZ камеры (трансляция)  →  PublicDisplay  ←  анимированные иконки
```

1. **YOLOv8n** — быстрый триггер на 25 камерах (есть ли лошади в кадре?)
2. **YOLOv8s** — точная детекция только на активных камерах
3. **ColorCNN** — классификация цвета торса жокея (5 цветов: красный, синий, зелёный, жёлтый, фиолетовый)
4. **4-уровневая фильтрация** — уверенность CNN, согласие CNN+HSV, ограничение скорости, временное подтверждение
5. **Голосование** — стабильный порядок определяется по итогам каждого видео

## Два интерфейса

- **`/`** — Публичный экран: PTZ-видео + таблица с анимированными иконками жокеев (стиль LONGINES)
- **`/operator`** — Панель оператора: сетка камер, 2D-трек, настройки гонки, переключение PTZ

## Ключевые особенности

- **25 аналитических + 4 PTZ камер** — строгое разделение: аналитика скрыта от зрителей
- **GPU-ускорение** — CUDA decode, TensorRT, опциональный DeepStream (C++)
- **4 языка** — турецкий, русский, казахский, английский (i18next)
- **Анимации** — плавное перемещение иконок, дуги обгона/отставания, цветовая подсветка
- **Docker** — контейнеризация всех сервисов (go2rtc, бэкенд, фронтенд)

## Запуск

```bash
# Бэкенд
python api/server.py --video video1.mp4 video2.mp4 --auto-start

# Фронтенд
cd Kabirhan-Frontend && npm install && npm run dev
```

Доступ: `localhost:5173` (фронт), `localhost:8000` (API), `ws://localhost:8000/ws` (WebSocket)
